Was sie benötigen:

  • Programmierfähiger Computer oder Laptop (Nicht im Lieferumgfang)
  • ESP 8266 (Im Lieferumfang)
  • BreadBoard (Im Lieferumfang)
  • DHT 11 Temperatursensor (Im Lieferumfang)
  • MQ-2 Gas Sensor (Im Lieferumfang)
  • TIP 120 Transistor (Im Lieferumfang)
  • Wiederstand (330Ω) (Im Lieferumfang)
  • Heizung/Lüfter (Im Lieferumfang)
  • Steckbrückenkabel (Im Lieferumfang)
  • USB A auf micro USB Kabel (Im Lieferumfang)

Was muss im Vorhinein gemacht werden?

Bevor Sie anfangen sollten Sie sichergehen, dass Sie die aktuellste Version der Arduino IDE installiert haben. Diese können Sie ganz einfach hier herunterladen. Nachdem Sie dies sichergestellt haben, müssen Sie noch einige Vorkehrungen im Programm treffen.

Gehen Sie zunächst einmal auf »File« -> »Prefrences«

Geben Sie nun http://arduino.esp8266.com/stable/package_esp8266com_index.json in das Feld Additional Boards Manager URLs ein und drücken Sie anschließend auf "OK"
Öffnen Sie anschließend den Boards Manager. Dazu müssen sie auf »Tools -> »Board« -> »Boards« »Manager«

Suchen Sie nun nach "ESP8266" und drücken Sie auf "Install"
Nach wenigen Sekunden sollten Sie Startklar sein.

Der Aufbau des Saunomats

Nehmen Sie die Bestandteile aus der Verpakung und ordnen Sie diese nach folgendem Schaltplan an: Verbinden Sie anschließend den ESP8266 via USB Kabel mit Ihrem Computer und starten Sie die Arduino IDE. Nun müssen Sie nurnoch den Code draufladen. Diesen downloaden Sie ganz einfach hier
